## Objective
Simulate a Transportation Operations Center (ROC) dashboard to track shipment performance and delivery metrics.

## Tools
- SQL (SQLite/MySQL/PostgreSQL)
- Power BI Desktop
- CSV files

## Features
- Aggregates shipments per region and staff
- Tracks KPIs: Total Shipments, On-Time Delivery %, Average Delay
- Visualizations: KPI Cards, Bar Chart, Line Chart, Tables
- Filters for Date, Region, Staff

## Usage
1. Load CSV files into Power BI
2. Create relationships between shipments and staff
3. Add DAX measures for on-time delivery % and avg delay hours
4. Build visuals as described above
